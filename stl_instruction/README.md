STL 分為三大部分，分別為 Container, Iterator 和 Alogrithm。

### Container

分為 Sequence (序列式), Associated (關聯式), Unordered (無序式)

* Sequence
    * [vector](container/vector.md)
    * [list](container/list.md)

* Associated
    * [map, multimap](container/map.md)

* Unordered

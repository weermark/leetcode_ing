## **普通模式 (normal-mode)(格光標)**

<br/>

### *切換*
指令|用途
---|---
v|切換可視模式，單格選取
V|切換可視模式，單行選取
`<crtl>` v|切換可視模式，單行列選取
i|切換編輯模式，光標移動到字前
a|切換編輯模式，光標移動到字後
I|切換編輯模式，光標移動到行首
A|切換編輯模式，光標移動到行尾
:|切換命令模式

<br/>

### *移動*
指令|用途
---|---
h|光標左移
j|光標下移
k|光標上移
l|光標右移
w
e

<br/>

### *編輯*

指令|用途
---|---
di{|刪除{}內的內容 (delete inside { })
da{|刪除{}內的內容，包含{}符號 (delete all { })
